
Plugin for [screpl](https://github.com/mighty1231/screpl)

required

* screpl in `lib` folder of euddraft

How to 

* Copy myapp.py and mymodule.py into `plugins` folder of euddraft


Make edd file should as follows

```
[repl]

[mymodule]

```

or as follow

```
[repl]

[mymodule]
command: openMine
```
